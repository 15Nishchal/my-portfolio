Face Detection Web Application

It is a web application. It provides the signin and register options.
A particular user can create their account by filling a simple registration form and can login with their credentials.
After login, they have the facility to copy an face-image url in the url search box and then the app can detect where the face is located in the image.

Each time they detect an image, their image count or entry count increases by 1.
This web app uses ReactJs, NodeJs, ExpressJs and Postgres SQL. Apart from the tech stack, it uses the Clarifai API's face-detection model to detect the face in the image.
